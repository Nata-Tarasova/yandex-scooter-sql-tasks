# yandex-scooter-sql-tasks
Выполнение практических заданий по работе с базами данных (PostgreSQL) в рамках тестирования бэкенда Яндекс Самоката. Включает в себя проверку отображения созданных заказов с использованием агрегатных функций, операторов связывания таблиц (JOIN) и условной логики (CASE WHEN) для проверки корректности записи статусов в БД.
